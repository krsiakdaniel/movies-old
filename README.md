### React

Run app on <http://localhost:3000/>

```
make start
```

Build the app in `/build/` folder.

```
make build
```

### Prettier

Changed files are formatted on save in editor and with each commit using [.prettierrc](.prettierrc) options. It can be done manually as well.

```
make prettier
```
